# Piano chord images

Put your saved piano chord images here. The app will display them when that chord is shown.

**File naming:** Use the chord name followed by the extension.

Examples:
- `C.png`
- `Cmaj7.png`
- `C#.png` or `Db.png` (same note; either filename works)
- `D#.png` or `Eb.png`
- `F#.png` or `Gb.png`, `G#.png` or `Ab.png`, `A#.png` or `Bb.png`
- `G7.png`, `Fmaj7.png`, etc.

Enharmonics are matched: if the song shows C# but you only have `Db.png` (or the other way around), the app will use it. If an image exists for a chord, it will be shown instead of the generated diagram; otherwise the app falls back to the built-in piano diagram.
